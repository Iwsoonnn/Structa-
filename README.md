# Dokumentacja projektu – STRUCTA

## 1. Informacje ogólne
Projekt STRUCTA to strona internetowa wykonana dla rzeczywistego klienta
w ramach indywidualnego projektu komercyjnego.

Strona została zaprojektowana i wdrożona jako wizytówka działalności
związanej z usługami technicznymi i projektowymi.

Strona dostępna online:
https://structa3d.com
